## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository by [Hitesh Suryawanshi](https://www.linkedin.com/in/hitesh-suryawanshi-147399111/).

## Installation

```bash
# install node_module
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Swager document

```bash
# build api doc
$ npm run build:doc
```
